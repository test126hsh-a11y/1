Этот код определяет функцию для обработки строки, содержащей числа, разделённые запятыми. Функция извлекает уникальные числа, сортирует их и выводит на экран вместе с их количеством. Также включена тестовая функция, которая проверяет корректность работы основной функции с различными входными данными. В конце кода вызывается основная функция с примером входной строки, что демонстрирует её работу.
